# Revision

Esta carpeta funciona como bandeja de entrada para materiales que todavia no
tienen una ubicacion definitiva dentro de `AulaTeX-Academico`.

## Estado actual

La bandeja quedo limpia el 2026-05-12. El material de
`template-latex.github.io` se integro como fuente editorial en:

```text
retroalimentacion-editorial/fuentes/template-latex-web/
```

## Uso sugerido

- Coloca aqui carpetas completas que quieras revisar antes de integrarlas.
- Conserva proyectos Slidev, recursos externos o experimentos sin mezclarlos con
  `trabajos/`.
- Cuando una carpeta ya tenga una logica clara, se mueve a `trabajos/`,
  `referencias/`, `herramientas/`, `plantillas/` o
  `retroalimentacion-editorial/fuentes/`.
- Git ignora el contenido de esta carpeta salvo este README y `INVENTARIO.md`.
