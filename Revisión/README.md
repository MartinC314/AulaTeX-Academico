# Revisión

Esta carpeta funciona como bandeja de entrada para materiales que todavia no tienen una ubicacion definitiva dentro de `AulaTeX-Academico`.

Uso sugerido:

- Coloca aqui carpetas completas que quieras revisar antes de integrarlas.
- Conserva proyectos Slidev, recursos externos o experimentos sin mezclarlos con `trabajos/`.
- Cuando una carpeta ya tenga una logica clara, se mueve a `trabajos/`, `referencias/`, `herramientas/` o `plantillas/`.
- Por seguridad, Git ignora el contenido de esta carpeta salvo este README y `INVENTARIO.md`.
