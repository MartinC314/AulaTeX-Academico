# Temas Selectos de Matematicas I

Esta materia puede trabajar informes y presentaciones en paralelo.

Estructura sugerida:

- `informes/`: fuentes `.tex` de reportes o informes.
- `presentaciones/`: presentaciones Beamer o, mas adelante, proyectos Slidev integrados.
- `recursos/`: imagenes, datos, capturas o materiales propios de la materia.

Plantillas relacionadas:

- Informe: `plantillas/latex/adaptadas/materias/temas-selectos-matematicas-1/plantilla-informe-temas-selectos-iiiepe.tex`
- Presentacion: `plantillas/latex/adaptadas/instituciones/iiiepe/plantilla-presentacion-iiiepe.tex`
