# Plantillas Adaptadas

Aqui viven los moldes que ya tienen decisiones personales, institucionales o de materia.

- `instituciones/`: formato general por institucion.
- `materias/`: derivaciones para una materia concreta.

Para crear una actividad nueva, copia el archivo de plantilla que corresponda hacia `trabajos/` y renombra la copia con materia, unidad, semana o actividad.
