# Catalogo de plantillas LaTeX

## Bases ppizarror

`ppizarror/originales/` contiene copias base descargadas de Template-Latex. Se conservan separadas para consulta y comparacion; no conviene editarlas directamente.

Plantillas integradas:

- `Template-Articulo`
- `Template-Auxiliares`
- `Template-Controles`
- `Template-Presentacion`
- `Template-Reporte`
- `Template-Tesis`
- `Template-Poster`
- `Professional-CV`

Todas tienen espacio en el catalogo. Algunas entraran al flujo cotidiano antes que otras, pero ninguna queda descartada.

## Adaptadas

`adaptadas/` contiene plantillas que si forman parte del flujo personal:

- `instituciones/unadm/plantilla-maestra-unadm.tex`
- `instituciones/iiiepe/plantilla-maestra-iiiepe.tex`
- `instituciones/iiiepe/plantilla-presentacion-iiiepe.tex`
- `materias/filosofia-derecho/plantilla-maestra-filosofia-derecho-unadm.tex`
- `materias/temas-selectos-matematicas-1/plantilla-informe-temas-selectos-iiiepe.tex`

Regla de trabajo: copiar una plantilla adaptada hacia `trabajos/` y editar la copia. Las plantillas base se modifican solo cuando se quiere cambiar el molde general.
